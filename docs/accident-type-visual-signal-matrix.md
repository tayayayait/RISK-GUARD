# 사고유형 시각신호 매트릭스 (18종)

## 목적
- 사고유형을 텍스트 없이도 즉시 식별할 수 있도록, 유형별 `MUST HAVE / MUST NOT HAVE` 시각신호를 고정한다.
- 본 문서는 `src/services/predictionService.ts` 프롬프트 규칙의 기준표다.

## 공통 원칙
- 모든 유형은 `사고 직전 1-2초` 장면만 허용한다.
- 이미지 내부 문자/아이콘/콜아웃/화살표 오버레이는 금지한다.
- `위험원 하이라이트 / 부상 가능 부위 / 위험 진행 방향 / 즉시 조치 포인트` 4신호를 동일 프레임에 포함한다.

## 유형별 기준
| 사고유형 | MUST HAVE | MUST NOT HAVE |
| --- | --- | --- |
| 끼임 | 회전체·이송부와 진입 간극, 신체 진입 경로 | 간극 식별 불가한 원경, 낙하물 중심 장면 |
| 협착 | 수렴하는 두 면과 압착 방향, 여유공간 상실 | 절단날 중심 장면, 단순 접촉 |
| 절단 | 활성 절단날/절단점, 절단 경로 | 비기계 충돌 중심 장면 |
| 베임 | 날카로운 모서리, 손/팔 접촉 경로 | 압궤/낙하 중심 장면 |
| 찔림 | 뾰족체 끝 방향, 관통 경로 | 평면 충돌 중심 장면 |
| 충돌 | 작업자-장비(또는 하중) 교차 동선, 제동 여유 부족 | 정지물 접촉 중심 장면 |
| 부딪힘 | 고정 구조물 모서리와 작업자 진행 방향 | 비래물 피격 중심 장면 |
| 맞음 | 이동 물체의 접근 궤적, 방어 자세 | 고정물 접촉 중심 장면 |
| 낙하·비래 | 낙하/비산 시작점, 궤적, 작업자 노출 위치 | 이미 충돌 완료된 결과 장면 |
| 추락 | 높이·개구부·가장자리, 중심 상실 자세 | 동일 높이 미끄러짐 중심 장면 |
| 넘어짐 | 바닥 위험원(오일/장애물/케이블), 발 디딤 오류 | 고소 추락 중심 장면 |
| 감전 | 충전부/활선, 접촉 근접 경로, 스파크/아크 | 기계적 충돌만 보이는 장면 |
| 화상 | 고온원/열원, 열 전달 방향, 노출 부위 | 폭발 압력파 중심 장면 |
| 폭발 | 압력·가연원과 점화원의 결합, 초기 폭발 전개 방향 | 단순 화염만 있는 장면 |
| 화재 | 점화원+가연물, 화염/연기 확산 방향 | 폭발파 중심 장면 |
| 깔림 | 중량물 하강/전도 경로, 하중 면적과 작업자 위치 | 경량 파편 접촉 중심 장면 |
| 압궤 | 고하중 압축원(프레스/하중), 압축 방향 | 단순 베임/접촉 중심 장면 |
| 붕괴 | 구조 불안정(기울어짐/균열/지지 실패), 붕괴 예상 경로 | 구조 안정 상태의 무관 위험 강조 |
