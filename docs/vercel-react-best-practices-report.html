위험 시나리오 영향 가능성 현 상태 통제 방법
과도한 청크 분할 요청 수 증가로 체감 성능 저하(워터폴) 낮음~중간 500k 초과 경고 제거, 주요 vendor 분리 완료 RUM/네트워크 waterfall 모니터링 유지
Suspense 경계 오설계 전체 화면 fallback 증가, UX 저하 낮음 라우트 단위 경계 분리 완료 신규 화면 추가 시 전역 경계 회귀 여부 점검
동적 import 실패(네트워크/번들 경로) PDF 내보내기 실패 낮음~중간 정적 혼합 import 문제는 제거됨 실패 fallback 메시지/재시도 로직 추가
Step/Provider 경계 변경 실수 단계 이동/권한 로직 깨짐 중간 현재 라우팅 로직은 유지됨 state-machine 회귀 테스트 유지, 단계 이동 E2E 추가
